Entendendo os conceitos

Por mais que algumas pessoas acham que o HTML é uma linguagem de programação, na verdade é uma linguagem de marcação e com isso temos diversos elementos para exibir a página para o usuário, tais como tags que ajudam a montar o esqueleto de nossa página. Se formos fazer uma analogia o HTML é o esqueleto do corpo humano, o CSS é o estilo e o JavaScript sria nossos orgãos para o funcionamento completo de uma página Web.

A estruturação do texto

(voltar)

A estrutura segue uma definição padronizada, qualquer site, blog ou página web, é composta de tags como "!DOCTYPE" que traz informações que serão carregadas antes da página ser exibida ao usuário, como o formato de codificação de caracteres usado na página, a compatibilidade e qual navegador esta sendo utilizado e também carrega o nome da página que pode ser modificada através da tag "title" e todas essas informações fazem parte do "head" que em tradução literal a cabeça da nossa página

Já dentro do "body" encontramos todas as tags e informações que serão exibidas para o usuário, os textos, as cores, as imagens.

Utilizando tags

(voltar)

As tags marcam o texto e estrutura de alguma forma, assim como no word existem, formas e tamanhos, espaçamentos para deixar as palavras em destaque, no html utilizamos tags para fazer essa modificação, então temos as tags de títulos que vai desde o "h1" até o "h6", as de parágrafos "p", o negrito "strong", o itálico "i", inclusão de links com a técnica de ancoragem "a href="#"", a "small" para inclusão de uma palavra curta, como se fosse um botão e diversos elementos que podemos incluir nesta página para deixar mais interessante para o usuário.

A tag "del" insere um corte no texto, desta forma exemplo, a "font" não é suportada no html5, temos também a tag "abbr" que adiciona o tracejado embaixo da palavra marcada, usada para abreviações, DIO
